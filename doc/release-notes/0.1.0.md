## libwallet 0.1.0

Initial versioned release of libwallet.

### Added

#### Wallet Management
- Create new wallets with encrypted seed storage
- Create watch-only wallets from extended public key
- Load and close existing wallets
- Change wallet passphrase with seed re-encryption
- Graceful shutdown handling

#### Seed & Recovery
- 12-word BIP39 seed support
- 15-word Decred seed support (with embedded birthday)
- 24-word BIP39 seed support
- Seed passphrase support (BIP39)
- Wallet recovery from mnemonic
- Restore from locally stored encrypted seed

#### Address Management
- Get current receive address
- Generate new external addresses
- List used and unused addresses
- Validate addresses
- Get default account public key
- Derive addresses from extended keys
- Create extended keys

#### Transactions
- Create and sign transactions
- Send-all functionality
- Broadcast raw transactions
- List transactions
- List unspent outputs
- Decode transaction hex
- Get transactions by hash
- Add signatures to unsigned transactions (for hardware wallets)
- Fee estimation via oracle

#### Synchronization
- SPV sync with configurable peers
- Sync status reporting
- Rescan from specific block height
- Rescan progress tracking
- Birth state (height/time) retrieval
- Best block retrieval

#### Message Operations
- Sign messages with wallet address
- Verify signed messages

#### CGO Interface
- JSON-encoded responses for all operations
- Error codes for structured error handling
- C-archive build support (`libdcrwallet.a`)

#### Logging
- Configurable log levels
- Rotating log files
- Optional log file disable
- Dual-output logging (file + error stream)

### Dependencies
- decred.org/dcrwallet/v5 v5.0.3
- decred.org/dcrdex v1.0.5
- github.com/decred/dcrd (various packages)
